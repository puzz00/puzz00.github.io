Google Dorks

We can use google search operators along with key words and in combination with other search operators to find out useful data about target websites, organisations, employees etc

A useful search operator is site:elsfoo.com as we can use it to find, amongst other things, subdomains of the target domain. We can do this more accurately using site:*.elsfoo.com

We can use the inurl:admin search operator to find interesting strings in urls.

We can use the intitle:passwd search operator to find interesting strings in webpage titles.

We can use the filetype:csv search operator to find files with specific file extensions.

The search operators can be used together to search in a more targeted way site:*elsfoo.com inurl:admin

A useful resource for google dorks can be found @ ghdb

Wayback Machine

We can see how websites used to look by using the wayback machine

It can be useful to look at historical versions of websites as we can gain a better understanding of the target and possibly find still useful data which is no longer available on the current version of a website.
